2. Предварительный анализ
Анализ корреляционной матрицы выявил наличие значимых линейных взаимосвязей между объясняющими переменными. Наибольшая корреляция наблюдается между alcohol и density (0.72). Распределение целевой переменной Y сбалансировано.

3. Снижение размерности методом PLS
Метод частных наименьших квадратов выделил 3 главные компоненты, объясняющие 82.4% дисперсии исходных переменных. Логистическая регрессия на преобразованном пространстве объясняющих переменных показывает точность 0.942 при перекрёстной проверке по 10 блокам.

4. Регуляризация методом лассо-регрессии
Подбор оптимального значения гиперпараметра C методом перекрёстной проверки дал результат C = 0.215. Модель лассо-регрессии обнулила 4 коэффициента из 11, выполнив отбор наиболее значимых переменных. Точность модели составляет 0.951. График сжатия параметров модели демонстрирует обнуление слабых коэффициентов при увеличении параметра регуляризации.

5. Выбор лучшей модели
Сравнение точности моделей показало преимущество лассо-регрессии (0.951) над PLS-моделью (0.942). Прогноз на отложенных наблюдениях подтвердил высокую точность лучшей модели - 0.960, с правильной классификацией 144 из 150 наблюдений.
